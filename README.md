# Cloud Event Platform

Internal cloud platform service for ingesting and querying operational events, designed with infrastructure-as-code and automated CI/CD.

## Project Goals
- Demonstrate cloud architecture design
- Reproducible infrastructure using IaC
- Automated deployment pipelines
- Operational considerations (costs, observability, tradeoffs)

## Project Structure
- `app/` – Service implementation
- `infra/` – Infrastructure as Code
- `docs/` – Architecture, decisions and documentation

## Current Status
- v0 – Project initialized and design defined
